🛡️ সিস্টেম নিরাপত্তার ৩ স্তর

স্তর ০১

Database Level

PostgreSQL Row-Level Security — এমনকি API bypass করলেও ডেটা বের হবে না।

স্তর ০২

Server Level

প্রতিটি action-এ requirePermission() check হয়।

স্তর ০৩

UI Level

Button / link-ই দেখা যাবে না যেগুলো ব্যবহারের অনুমতি নেই।

✓ ফলাফল: একজন Accountant কখনও exam marks দেখতে বা change করতে পারবে না — এমনকি URL manipulate করলেও না।
